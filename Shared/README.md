# IFCDC Shared Assets

Cross-project shared resources, configurations, and utilities.

## Contents

- Brand guidelines and design tokens
- Common environment variable templates
- Shared TypeScript configurations
- Cross-app utility scripts

## Usage

Reference shared assets from any IFCDC application:

```typescript
// Import shared types or configs as needed
import { IFCDC_BRAND } from "@ifcdc/ui-components";
```

## Related

Shared code libraries live in `Libraries/ifcdc-packages/`.
